Api Besar Keluar di Madinah  Benarkah bagian dari tanda kiamat, ada api besar di arab? Berarti api kilang minyak itu tanda kiamat?  Jawab:  Bismillah was shalatu was salamu ala Rasulillah, wa ba'du,  Api yang keluar di akhir zaman ada 2:  Pertama, api besar yang keluar di daerah Hijaz (tepatnya di Madinah). Api ini sudah keluar di masa silam, tahun 654 H.  Kedua, api yang keluar dari Yaman, yang akan menggiring manusia ke Syam  Yang dimaksud dalam pertanyaan adalah api yang keluar dari Madinah.  Mengenai keberadaan api besar ini, pernah disabdakan oleh Nabi Shallallahu ‘alaihi wa sallam dalam hadis yang diriwayatkan Bukhari & Muslim.
